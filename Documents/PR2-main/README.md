# Practical Research Cybersecurity Hub

This website helps students learn the basics of cybersecurity through a simple, student-friendly dashboard.

It includes:
- A searchable list of cybersecurity topics,
- Easy-to-understand explanations for each topic,
- Links to trusted resources for more information,
- A clean frontend interface built with HTML, CSS, and JavaScript.

The site is designed for students to explore security concepts, understand safe online behavior, and access practical guidance for school accounts and devices.
